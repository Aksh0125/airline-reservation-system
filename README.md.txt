# Airline Reservation System

A full-stack Airline Reservation System developed using C# .NET, Java, and MySQL.

## Features
- User & Admin authentication
- Flight booking and cancellation
- Seat availability management
- MySQL database integration

## Tech Stack
- Backend: C# .NET, Java
- Database: MySQL

## Project Structure
- backend-dotnet → C# backend
- backend-java → Java services
- database → SQL scripts

## How to Run
1. Import SQL file from `/database`
2. Configure DB connection
3. Run backend services
